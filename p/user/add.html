我是参考鸟哥的Linux创建的

新建用户并且指定密码

      
        [root@localhost ~]# useradd czm
        [root@localhost ~]# passwd czm
        输入两次密码

添加sudo能力

        
        [root@localhost ~]# vim /etc/sudoers
        ## Allow root to run any commands anywhere
        czm     ALL=(ALL)       ALL
